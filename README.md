## 🛠️ Tech Stack

* **Language**: Swift
* **Framework**: SwiftUI (Declarative UI layout)
* **Core Libraries**: AVFoundation (Hardware audio activation & playback)
* **IDEs & Tools**: Xcode 17+ / Git Version Control

## 📂 Project Directory Structure

* `MyViews/` - Holds decoupled, reusable subviews (gestures, counters, hands).
* `Utilities/` - Manages the background and sound effects playback engine.
